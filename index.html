☁️ 온라인 동기화 설정

📋 설정 방법

  1. GitHub 계정으로 로그인 (github.com)
  2. Settings → Developer settings → Personal access tokens → Tokens (classic)
  3. "Generate new token (classic)" 클릭
  4. Note: "Minah Teacher Points", Expiration: "No expiration"
  5. Scopes에서 "gist" 체크박스만 선택
  6. 생성된 토큰을 아래에 입력

토큰은 안전하게 암호화되어 저장됩니다.
